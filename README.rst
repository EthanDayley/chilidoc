ChiliDoc
=========

This package is designed to provide an AsciiDoc-like markdown language
which can be compiled to "pure" html.


:Authors:
    Ethan Dayley
:Version:
    1.0.0

Installation
------------

Install with pip: `pip install chilidoc`

Usage
------

::
    Import chilidoc
    chilidoc.to_html(your_chilidoc_string)
